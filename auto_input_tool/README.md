# 自动输入工具

这个工具可以帮助你自动输入文本到禁止粘贴的输入框中。

## 安装依赖

```bash
pip install pyautogui
```

## 使用方法

1. 运行脚本:
```bash
python auto_input.py
```

2. 在文本框中输入内容，或点击"从文件加载"按钮导入txt文件

3. 点击"开始输入(3秒后)"按钮

4. 3秒内切换到目标输入框

5. 如需停止，点击"停止"按钮

## 功能特点

- 图形界面操作
- 支持停止按钮
- 支持中文输入
- 支持从文件批量读取
- 实时显示输入进度
- 可调节输入速度

## 调节速度

在代码中修改 `time.sleep(0.03)` 中的数值可以调节输入速度:
- `0.01` - 快速
- `0.03` - 中速
- `0.1` - 慢速
