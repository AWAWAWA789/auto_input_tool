# 自动输入工具 Pro

一个支持暂停/继续/自选起始位置的自动输入工具，适用于需要自动输入文本但平台禁止复制粘贴的场景。

## 功能特点

- 🚀 自动逐字符输入文本
- ⏸️ 支持暂停和继续输入
- 🔄 支持从头输入
- 📍 支持自选起始位置
- 📊 实时进度显示
- 🖱️ 鼠标点击自动停止
- 📁 支持多种编码的txt文件导入
- 🎯 停止位置高亮显示

## 安装依赖

```bash
pip install pyautogui pyperclip pynput
```

或使用 requirements.txt：

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动程序

```bash
python auto_input.py
```

### 2. 输入内容

- 直接在文本框中输入内容
- 或点击「从文件加载」按钮导入txt文件

### 3. 开始输入

- 点击「开始输入」按钮
- 3秒倒计时后自动开始输入
- 输入过程中，点击鼠标任意键可立即停止

### 4. 暂停与继续

- 输入过程中点击「停止」按钮或点击鼠标暂停
- 暂停后点击「继续」按钮可从停止位置继续输入
- 暂停后点击「从头输入」按钮可从头开始输入

### 5. 自选起始位置

- 在停止状态下，点击文本框中的任意位置
- 点击「设为起点」按钮确认起始位置
- 点击「开始输入」或「继续」从设置的位置开始输入

## 界面说明

| 按钮 | 功能 |
|------|------|
| 开始输入 | 从头开始输入（3秒倒计时） |
| 停止 | 暂停当前输入 |
| 继续 | 从停止位置继续输入 |
| 从头输入 | 从第一个字符重新开始 |
| 设为起点 | 将点击位置设为起始位置 |
| 从文件加载 | 导入txt文件 |

## 快捷键

- 无（通过鼠标操作控制）

## 注意事项

1. 输入过程中点击鼠标任意键会立即停止输入
2. 支持UTF-8、GBK、GB2312等多种编码格式
3. 输入时会临时使用剪贴板，输入完成后自动恢复
4. 建议在输入前先点击目标输入框
